# Customized keyboard from scratch!

## Requirements
- LCD display
- knob
- rgb backlit
- somehow layout the normal 60% plus pgup, pgdn, del, arrows, lcd, and knob and find compatible keycaps
- keycaps need to be, shine-through, square,
